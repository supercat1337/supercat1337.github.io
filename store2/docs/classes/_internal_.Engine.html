getMessage

  • getMessage(message, ctx): void

  • Processes a message that has been sent to the engine. If the message is DEPENDENCY_CHANGED, the engine notifies all dependents of the change. If the message is DEPENDENCY_DESTROYED, the engine destroys itself. If the message is HAS_ERROR, the engine forwards the error to all dependents.
